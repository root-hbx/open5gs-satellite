# Contributing to Open5GS-Satellite

Thank you for your interest in contributing to Open5GS-Satellite! We welcome and value all contributions to this project, including but not limited to:

- [Bug reports](https://github.com/root-hbx/open5gs-satellite/issues)
- [Pull requests](https://github.com/root-hbx/open5gs-satellite/pulls) for bug fixes and new features

**Contributing Code**

We use GitHub to track issues and features. For new contributors, we recommend looking at issues labeled "[good first issue](https://github.com/root-hbx/open5gs-satellite/labels/good%20first%20issue)".

**Submitting pull requests**

1. Fork the `open5gs-satellite` repository and create a new branch for your changes.
2. Follow the [Google style guide (Markdown)](https://google.github.io/styleguide/docguide/style.html).
3. Push your changes to your fork and open a pull request in the Open5GS-Satellite repository.
4. Delete your branch after merging it. This keeps the repo clean and faster to sync.
